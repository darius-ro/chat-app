# Chat App
A simple app that lets you communicate to anyone on the platform.

# Goals
- [ ]  Seperate frontend and backend server.
- [ ] Authentication (Login, Register, Reset Password)
- [ ] Send messages/images
- [ ] Reply to messages
- [ ] React to messages/images
- [ ] Captchas for bot prevention
- [ ] Rate limits to important routes
- [ ] Edit/Delete messages
